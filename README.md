<h3 align="center">A Task Manager app for organizing your works and todos</h3><br>


<h3 align="left">Used Techs:</h3>

- Flutter
- REST API


<h3 align="left">Features:</h3>

- User Authentication
- Otp Verification
- All tasks
- Running tasks
- Completed tasks
- Add task
- Delete task
- Chage status of tasks
- Update profile
